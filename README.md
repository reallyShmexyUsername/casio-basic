# casio-basic
Emulating the casio basic programming language

# Functionality

## Arithmetic

### Add
```python
expression + expression
```

### Subtract
```python
expression - expression
```

### Multiply
```python
expression * expression
```

### Divide
```python
expression / expression
```

## Logic

### Less than
```python
expression < expression
```

### Greater than
```python
expression > expression
```

### Equal to
```python
expression = expression
```

## Conditions

### And
```python
expression and expression
```

### Or
```python
expression or expression
```

### Not
```python
not expression
```

## Variables
### Variable Names
```regex
[A-Z]
```

### Variable Assign
```
expression -> variable_name
```
